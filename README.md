# Project Graph Plugin

Plugin globale per AI (Claude Code, GPT, Cursor, VSCode, JetBrains).

Skill disponibili:

- /organization-graph → genera struttura del progetto
- /memory-graph → carica memoria del progetto e la fissa in .claude/rules/project-graph.md (auto-caricato ad ogni nuova chat)
- /mem-modif → registra modifiche git

Installazione:
Copiare la cartella in:
%USERPROFILE%/.ai-global-plugins/project-graph
