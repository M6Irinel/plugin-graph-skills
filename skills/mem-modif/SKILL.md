---
name: mem-modif
description: Registra le modifiche git correnti (working tree/staged) in mem-modif.md nella root del progetto. Usa quando l'utente vuole tracciare o salvare uno storico delle modifiche recenti.
allowed-tools: Bash
---

Run `powershell -ExecutionPolicy Bypass -File "${CLAUDE_PLUGIN_ROOT}/commands/mem-modif.ps1"` in the project root e riporta cosa è stato registrato.
